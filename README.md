##ЕДИНСТВО НАРОДОВ РОССИИ

**Квест-игра о единстве народов России**  
*Разработано для гейм-дев хакатона*

## **ОПИСАНИЕ**

Вы — ученик мастера-хранителя древних артефактов. Ваша задача — восстановить 5 священных артефактов, принадлежащих разным народам России. Для этого вам предстоит отправиться в путешествие, выполнить задания и узнать традиции каждого народа.
## **ЛОКАЦИИ И НАРОДЫ**

| Локация | Народ | Артефакт | Задание |
|---------|-------|----------|---------|
| **Татарская деревня** | Татары | Амулет с полумесяцем | Собрать 3 детали для телеги к Сабантую |
| **Русская кузница** | Русские | Подкова | Выковать подкову (мини-игра Z/X/C) |
| **Сибирская тайга** | Буряты | Шаманский бубен | Поймать рыбу (мини-игра L/K) |
| **Северная тундра** | Ненцы | Оберег из клыка | Найти оленёнка по тотемам, согреть у костра |
| **Кавказские горы** | Осетины | Рог изобилия | Донести чашу старейшине через сломанный мост |

## **ОСНОВНОЙ ГЕЙМПЛЕЙ**

1. **Старт в мастерской** — поговорить с Мастером (E)
2. **Выбрать артефакт** — подойти и нажать F
3. **Телепорт** в локацию народа
4. **Поговорить с главой** — получить задание (E)
5. **Выполнить мини-игру**
6. **Получить кристалл**
7. **Вернуться в мастерскую**
8. **Повторить 5 раз**
9. **Финальная речь Мастера** и титры

## **УПРАВЛЕНИЕ**

| Клавиша | Действие |
|---------|----------|
| **WASD** | Передвижение |
| **Мышь** | Поворот камеры |
| **E** | Взаимодействие / диалог |
| **F** | Использовать / взять предмет |
| **Пробел** | Продолжить диалог / согреть оленёнка |
| **Z, X, C** | Ковка подковы |
| **L / K** | Управление зоной в рыбалке |
| **ESC** | Меню выхода |
| **TAB** | Дневник путешественника |

## **МИНИ-ИГРЫ**

### **Русская кузница**
- Запомни последовательность (Z, X, C)
- 10 раундов сложности

### **Сибирская рыбалка**
- Удерживай рыбу в зелёной зоне
- Управление: L (влево), K (вправо)
- Продержаться 30 секунд

### **Северный оленёнок**
- Найди 3 тотема (F)
- Подбери оленёнка
- Согрей у костра (Пробел)

### **Кавказский мост**
- Возьми чашу у башни
- Пройди по мосту 
- Донеси старейшине

### **Татарская деревня**
- Собери 3 детали
- Простое собирательство

## **НАСТРОЙКИ**

-  **Громкость** — ползунок
-  **Разрешение экрана** — выбор предложенных вариантов

## **ФИНАЛ**

После активации всех 5 артефактов:
1. Мастер меняет цвет
2. Финальная речь о единстве народов
3. Титры с благодарностями
4. Возврат в главное меню
